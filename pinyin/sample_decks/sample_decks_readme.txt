This distribution includes two sample decks for you to use and experiment with.
The fields, models, and fonts have been pre-configured for optimal use with Pinyin Toolkit.

BlankDeckSimple contains 2 main testing models, plus measure word testing.
It is thus more similar to the default Anki decks.

Blank Advanced contains 5 main testing models, pluts measure word testing.
It is a more advanced version of deck that seperates each type of testing.
See: http://www.supermemo.com/articles/20rules.htm#minimum information principle

Text-to-speech audio will be added automatically once you set it up as per the readme file.

You should wipe the sample cards before use (9 and 21 cards respectively).

The zip file contains the original copies of the decks in case you make accidental changes.
